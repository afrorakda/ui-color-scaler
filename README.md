# #87 UI Color Scaler

A minimalist tone generator for UI designers and developers. Create logically consistent color variations from a single base color in one second.

## 1 Tool / 1 Action
- **Input:** Type a 6-digit HEX code or select a color using the visual picker.
- **Action:** The tool instantly generates a brightness scale and its complementary color.
- **Output:** Tap any color card to copy the HEX code to your clipboard.

## Features
- **Immutable # Prefix:** Focus only on the 6-digit code. The "#" is fixed to prevent input errors and double-pasting issues.
- **Brightness Scale:** Generates 6 logical steps from 95% (tint) to 5% (shade) lightness, perfect for backgrounds, hovers, and borders.
- **Contrast Indicator (●):** Each color card features a dot that automatically switches between Black and White based on WCAG luminance, showing you the best text color for that background.
- **Complementary Tone:** Instantly finds the opposite color (180° hue shift) for use as a UI accent.
- **Mobile Optimized:** Features a specialized numeric/alpha keyboard trigger and large touch targets for effortless use on the go.

## Design System
- **Background:** `#fff`
- **Main Color:** `#333`
- **Border Radius:** `15px` (Mandatory)
- **Visual Picker:** High-visibility, independent button for intuitive color selection.
